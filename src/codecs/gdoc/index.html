gdoc

A codec for Google Documents (GDoc).

This codec decodes from, and encodes to, the GDoc JSON ('application/vnd.google-apps.document') as defined as JSON Schemas here and in Typescript here

See also the guide on the structure of a GDoc.

It is possible to get this JSON using the Google Docs API get method. However, it is not possible to POST i.e. upload the JSON. The create method only creates a*blank** document:

Creates a blank document using the title given in the request. Other fields in the request,*including any provided content, are ignored**.

There are two other possibilities for creating a GDoc from content:

  1. Use the Google Drive API import method to upload content such as HTML.
  2. Use Google App Script API to walk Stencila JSON and build up the GDoc by calling methods such as Body.appendParagraph etc.
