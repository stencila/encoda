Const whiteSpaceRegEx

whiteSpaceRegEx: RegExp = new RegExp(/(^|[^\n\s])[\n]+\s*(?![\n])/g)
  • Defined in src/codecs/md/index.ts:94

Matches new lines, the preceding character, and any following white space on the next line. This so so that trailing spaces can be collapsed into a space. To use you need to include the matched group in the replacement value.

see

https://regexr.com/4i45o

see

https://stackoverflow.com/a/18012521

example

myString.replace(whiteSpaceRegEx, '$1 ')
